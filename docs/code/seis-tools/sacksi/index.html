    sacksi #

    获取PKIKP 射线与地球旋转轴之间的角度 \(\xi\)

    上图是PKIKP 射线路径在内核中的几何形状。地球的旋转轴用一条指向南北方向的虚线表示。PKIKP 射线与地球旋转轴之间的角度被定义为射线角（ξ）。

    假设PKIKP 波震源位置为 \(S(\theta_1,\phi_1)\) , 台站的位置为 \(R(\theta_2,\phi_2)\) 。 其中 \(\theta_1\) 为震源的余纬度(90º - 纬度), \(\phi_2\) 为震源的经度, \(\theta_2\) 和 \(\phi_2\) 分别是台站的余纬度和经度。

    计算公式如下：

    \[ \cos\xi = \frac{\left| \cos\theta_1 - \cos\theta_2 \right|}{\sqrt{2 - 2\cos\theta_1\cos\theta_2 - 2\sin\theta_1\sin\theta_2\cos(\phi_1 - \phi_2)}} \]

    在上述公式中，ξ 定义在0 - 90°区间内。对于赤道平面的射线路径，ξ= 90º，而穿过地球的射线路径与地球自旋轴平行，ξ= 0º。当ξ < 35º 时，相应的PKIKP 射线路径一般被称为南北向或南北向路径；反之则被称为东西向或东西向路径。
