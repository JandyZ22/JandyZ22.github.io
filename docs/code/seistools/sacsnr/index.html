    sacsnr #

    获取指定时间窗口内SAC文件的信噪比。

    公式如下：

    \[ S N R=10 \log _{10}\left(\frac{A_s}{A_N}\right)^2 \]

    Example # 

    (jandy) [ydz@gaia YL]$ sacsnr
successfully send 2005.06.01_2005.07.01
(jandy) [ydz@gaia YL]$
